# Instructions to clone this repo

## Steps 
  1. git clone https://github.com/mohamedaflah/taskmngment_task.git
  2. cd client
  3. npm install
  4. npm run dev
  5. cd ../server/
  6. npm install
  7. npm run build
  8. npm start

## Ensure .env file added in both client and server, refer .env.local and env.sample.txt file
